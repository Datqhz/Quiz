## Getting started
### Installation
1. **Pull docker images**
    - Navigate to `run-guest` directory
    ```
    cd run-guest
    ```
    - Pull all docker images
    ```
    docker-compose pull
    ```
2. **Build docker images**
    ```
    docker-compose build
    ```
3. **Setup database**
    ```
    ```
4. **Run all images as container**
    ```
    docker-compose up
    ```
